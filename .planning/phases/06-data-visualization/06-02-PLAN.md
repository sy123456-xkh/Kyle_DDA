# Phase 6 Plan 02: 图表配置面板

## 目标

创建完整的图表配置面板，允许用户切换图表类型、选择轴字段、自定义标题、切换颜色主题和图例位置。

## 前置条件

Plan 01 已完成：ChartView 组件可用，workspace 有 chartOverrides state。

---

## Task 1: 创建 ChartConfig 组件

**创建** `frontend/app/components/ChartConfig.tsx`

Props:
```typescript
// ChartOverrides 和 COLOR_THEMES 从 ChartView.tsx 导入
import { ChartOverrides, COLOR_THEMES } from './ChartView'

interface ChartConfigProps {
  spec: ChartSpec              // 后端原始 spec
  columns: string[]            // 查询结果列名
  overrides: ChartOverrides    // 当前覆盖值
  onChange: (overrides: ChartOverrides) => void
}
```

实现：
- 从 ChartView 导入 ChartOverrides 类型和 COLOR_THEMES 常量（共享色板定义）
- **图表类型切换：** 三个图标按钮（line/bar/pie），当前选中高亮
  - 图标用简单 SVG path：折线/柱状/饼图
  - 选中态：amber 背景 + 白色图标
- **X 轴字段：** select 下拉，选项来自 columns prop
  - pie 模式下标签改为"分类字段"
- **Y 轴字段：** select 下拉，选项来自 columns prop
  - pie 模式下标签改为"数值字段"
- **标题：** text input，placeholder 为 spec.title
- **颜色主题：** 4 个色块按钮
  - 使用 ChartView 导出的 COLOR_THEMES 常量（indigo/amber/emerald/rose）
  - 每个按钮显示该主题的首色作为色块
- **图例位置：** 4 个方向按钮（上/下/左/右）
  - 用箭头图标表示

布局：
- 折叠面板样式，默认收起，点击"图表配置"展开
- 展开后分区：类型 | 字段 | 样式（标题+颜色+图例）
- 紧凑布局，每行一个配置项，label 在左 control 在右
- 保持暗色主题风格

onChange 行为：
- 每次配置变更立即调用 onChange，传入完整 overrides 对象
- 类型切换时保留其他配置
- 字段切换时保留类型和样式配置

验证：
- `cd frontend && npx tsc --noEmit` 无类型错误
- 组件独立渲染，各控件可交互

完成标准：
- 6 个配置项全部可用（类型/X轴/Y轴/标题/颜色/图例）
- 折叠/展开动画流畅
- 暗色主题视觉一致

## Task 2: workspace 集成配置面板

**修改** `frontend/app/workspace/page.tsx`
**修改** `frontend/app/components/ChartView.tsx`

实现：
- 导入 ChartConfig 组件
- 在图表区域（ChartView）下方添加 ChartConfig
- 传入 props：
  - spec: activeResult.chart
  - columns: Object.keys(activeResult.rows[0]) （从查询结果提取列名）
  - overrides: chartOverrides（Plan 01 已创建的 state）
  - onChange: setChartOverrides
- 修改 ChartView：根据 overrides.colorTheme 选择 COLOR_THEMES 中对应色板应用到 series
- 修改 ChartView：根据 overrides.legendPosition 设置 ECharts legend 配置（orient + position）
- 当 activeResult 变化时（新查询），重置 chartOverrides 为 {}

验证：
- `cd frontend && npx tsc --noEmit` 无类型错误
- 上传 CSV → 执行 playbook → 图表显示 → 展开配置面板
- 切换类型：line ↔ bar ↔ pie 实时切换，不重新请求
- 切换 X/Y 字段：图表数据重新映射
- 修改标题：图表标题实时更新
- 切换颜色主题：图表颜色变化
- 切换图例位置：图例移动

完成标准：
- 配置面板在图表下方可折叠展示
- 所有配置变更实时反映到图表
- 新查询自动重置配置
